JavaScript: Scope và Closure

tháng 2 20, 2021
Scope là gì? Scope là phạm vi truy cập của một biến hoặc hàm từ một vị trí xác định trong code. Hiện nay, JavaScript có 3 loại scope: Global Scope: Biến được khai báo global (bên ngoài bất cứ function nào) thì sẽ có Global Scope. Biến global có thể được truy cập từ bất cứ đâu trong chương trình Javascript. Function Scope: Biến được khai báo cục bộ (bên trong một function) thì sẽ có Function Scope.

Với biến (khai báo bằng var): JavaScript sẽ chỉ chuyển phần khai báo biến lên đầu (tức phần gán giá trị vẫn giữ nguyên thứ tự) và gán cho giá trị undefined. Trong function sayHi(), chúng ta khai báo biến name với từ khóa var => hoisting đã xảy ra, JavaScript đã chuyển phần khai báo biến lên đầu và gán giá trị cho name là undefined.
